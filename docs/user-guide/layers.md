# Layers

Built-in layers:

- `IdealPointLayer`
- `PublicOpinionLayer`
- `LobbyingLayer`
- `MediaPressureLayer`
- `PartyDisciplineLayer`
- `GovernmentAgendaLayer`

Optional:

- neural layer support (`include_neural` + custom factory).

## Custom composition

Use `layer_names` and `layer_overrides` to build an explicit layer pipeline.
